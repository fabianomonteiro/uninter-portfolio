Portfólio

Aqui estão alguns dos meus trabalhos e projetos mais desafiadores:

  • Compile Systems for IoT

    Atualmente estou trabalhando no desenvolvimento de uma solução de IoT (Internet das Coisas). https://iot.compilesystems.com.br

  • Digitalmed (2015):

    Criação de uma solução para a transição tecnológica dos navegadores que removiam compatibilidade com plugins externos. Desenvolvi um aplicativo desktop que utilizava WebSocket via SignalR para permitir comunicação entre a aplicação web e hardwares como scanners e impressoras, resolvendo um problema crítico para o negócio.

  • Gorilla Telemetria e Rastreamento (2015-2018):

    Desenvolvimento de uma solução de telemetria para monitoramento de sensores em máquinas industriais, utilizando comunicação via protocolo Modbus e integração com um módulo de comunicação GSM. Criação de uma interface de gestão de dados para monitoramento em tempo real, auxiliando na atuação proativa no pós-venda.

  • DBCorp ERP (2018-2020):

    Reestruturação da arquitetura de software, implementando Clean Architecture, DDD, SOLID e Design Patterns. Transformação do desenvolvimento síncrono para assíncrono, adoção de boas práticas de desenvolvimento e criação de uma API pública para integração com E-Commerces externos. Desenvolvimento de um novo ERP Web utilizando VueJs e Quasar Framework.
